# Maha Events Co

A one-page marketing site for a wedding & proposal decor studio.

## Stack

Next.js (App Router) · TypeScript · Tailwind CSS v4 — no animation or form libraries; interactivity is plain React + IntersectionObserver + CSS transitions.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `lib/content.ts` — all site copy (services, gallery captions, testimonials) as typed data
- `components/reveal.tsx` — scroll-into-view fade/rise wrapper, used across every section
- `components/gallery-lightbox.tsx` — gallery grid + click-to-enlarge lightbox (arrow keys, Escape, focus return)
- `components/inquiry-form.tsx` — contact form with blur validation and a simulated submit/success state
- `public/images/` — placeholder decor illustrations generated by `scripts/gen-images.mjs`

## Notes

- The inquiry form simulates submission (no backend) and shows an inline success state.
- Placeholder images are generated locally (no external image host dependency) — regenerate with `node scripts/gen-images.mjs` after editing the palette, or replace files under `public/images/` with real photography using the same filenames.
