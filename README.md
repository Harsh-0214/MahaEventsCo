# Maha Events Co

One-page marketing site for Maha Events Co — wedding, South Asian celebration, and proposal decor styling. Built by [Strive](https://striveweb.ca).

## Stack

Next.js (App Router) · TypeScript · Tailwind CSS v4 · react-hook-form + zod

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `app/` — routes, root layout, global styles
- `components/` — page sections (`components/sections/`), shared UI primitives (`components/ui/`), and the `Reveal` scroll-animation wrapper
- `lib/content.ts` — all site copy and structured content (services, gallery, testimonials, nav, contact info) — add a new service or testimonial here, not in markup
- `lib/inquiry-schema.ts` — the zod schema backing the inquiry form; reuse this as the API contract when a real backend is wired up
- `public/images/` — placeholder imagery (generated by `scripts/gen-placeholders.mjs`); swap in real photography under the same filenames when available

## Notes

- The inquiry form simulates submission (no backend yet) and shows an inline success state.
- Placeholder images are generated locally and carry no external dependency — regenerate with `node scripts/gen-placeholders.mjs` if you tweak the palette.
